O pipeline que montei faz a leitura de várias abas do Excel, padroniza e limpa os dados, removendo inconsistências como valores ‘...’. Depois, analisei a distribuição de frequência dos óbitos e da taxa de mortalidade para entender o comportamento geral da base. Em seguida, apliquei uma Regressão Linear Simples para identificar a tendência histórica dos óbitos em relação ao ano. Por fim, usei estatísticas descritivas para resumir a evolução da mortalidade no trânsito. Esse processo forma a base para análises mais avançadas no futuro.
